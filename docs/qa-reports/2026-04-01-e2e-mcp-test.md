## Recomendações

### Imediato — Corrigir MCP Server (mcp-server/)
1. **Criar migration** adicionando as 6 colunas ausentes OU remover referências do código MCP às colunas que não existem
2. **Criar função `execute_readonly_query`** no Supabase (script no erro BUG-02)
3. **Corrigir `croma_criar_proposta`**: insert de itens deve ser atômico — se falhar, fazer rollback da proposta inteira
4. **Corrigir race condition** no número de proposta: usar `SELECT MAX(numero)+1` dentro de transação ou sequence dedicada
