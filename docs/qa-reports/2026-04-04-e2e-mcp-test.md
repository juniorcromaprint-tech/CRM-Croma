## Gaps do MCP / Schema

| # | Operação | Detalhe | Prioridade |

| GAP-SCHEMA-02 | Tabela instalações | E2E skill referencia `instalacoes` → correto é `ordens_instalacao` | 🔴 CORRIGIR NO SKILL |

| GAP-SCHEMA-04 | Tabela contas_receber | E2E skill referencia `contas_receber.valor` → correto é `valor_original` | 🔴 CORRIGIR NO SKILL |
| GAP-SCHEMA-05 | Tabela ordens_producao | E2E skill referencia coluna `prazo_entrega` → correto é `prazo_interno` | 🔴 CORRIGIR NO SKILL |
| GAP-MCP-04 | Conversão Lead→Cliente | Sem vínculo automático ao cadastrar cliente com mesmo email. Histórico acumula clientes duplicados por email | 🟡 MELHORIA FUTURA |
| GAP-INFO-01 | CR status=previsto | `totalizadores.total_aberto=0` com CR em status `previsto` — calculador não considera status `previsto` como aberto | 🟡 INVESTIGAR |
